# 小红书前端项目

一个仿照小红书探索页面的Vue3前端项目。

## 项目概述

这是一个使用Vue3 + Vite + ElementPlus构建的前端项目，模拟了小红书的探索页面，包含：

- 🎨 现代化的UI设计，高度还原小红书界面
- 📱 响应式布局，支持移动端和桌面端
- 🔄 无限滚动加载
- 🎯 分类筛选功能
- 💅 优雅的动画效果

## 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 快速的构建工具
- **Vue Router 4** - 路由管理
- **Pinia** - 状态管理
- **Element Plus** - UI组件库
- **Sass** - CSS预处理器
- **Axios** - HTTP请求库

## 快速开始


### 开发环境运行

```bash

```

项目将在 http://localhost:3000 启动

### 生产环境构建

```bash
npm run build
```

### 预览构建结果

```bash

```

## 功能特性

### 1. 探索页面 (Explore)
- **瀑布流布局**: 采用CSS Grid实现响应式瀑布流布局
- **无限滚动**: 滚动到底部自动加载更多内容
- **分类筛选**: 支持按时尚、美妆、美食、旅行、生活等分类筛选
- **帖子卡片**: 包含图片、标题、作者信息和互动数据

### 2. 顶部导航栏 (Header)
- **品牌Logo**: 小红书品牌标识
- **搜索功能**: 支持搜索用户、笔记、商品
- **导航菜单**: 探索、关注、消息、个人中心
- **发布按钮**: 快速发布新内容
- **用户头像**: 个人中心入口

### 3. 侧边栏 (Sidebar)
- **推荐关注**: 展示推荐用户，支持一键关注
- **热门话题**: 显示当前热门话题标签
- **推荐频道**: 按分类推荐相关内容频道

### 4. 响应式设计
- **桌面端**: 多列瀑布流布局，完整侧边栏
- **平板端**: 减少列数，隐藏侧边栏
- **移动端**: 双列布局，优化触摸体验

## 组件说明

### Header.vue
顶部导航栏组件，包含：
- Logo和品牌标识
- 搜索框
- 主导航菜单
- 用户操作区域

### PostCard.vue
帖子卡片组件，展示：
- 图片内容（支持懒加载）
- 帖子标题
- 作者信息
- 互动数据（点赞、评论、收藏）
- 悬停效果和动画

### Sidebar.vue
侧边栏组件，包含：
- 推荐用户列表
- 热门话题标签
- 推荐内容频道

### Explore.vue
主页面组件，整合：
- 分类筛选器
- 帖子瀑布流
- 无限滚动加载
- 响应式布局

## 样式设计

### 颜色方案
- **主色调**: #ff2442 (小红书红)
- **背景色**: #f8f8f8 (浅灰色背景)
- **文字色**: #333, #666, #999 (层级文字颜色)
- **边框色**: #e5e5e5, #f0f0f0 (分割线颜色)

### 设计原则
- **一致性**: 保持视觉风格统一
- **简洁性**: 去除冗余元素
- **可用性**: 优化用户体验
- **响应式**: 适配多端设备

## 性能优化

1. **图片懒加载**: 减少初始加载时间
2. **虚拟滚动**: 优化大量数据渲染性能
3. **组件缓存**: 避免重复渲染
4. **代码分割**: 按需加载组件
